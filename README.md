# 📌 Project Overview

This project focuses on analyzing **Brain Tumor Patient Data** using **Power BI** to uncover meaningful healthcare insights from a dataset containing **10,000 patient records**.

The dashboard transforms raw healthcare data into visually engaging and actionable healthcare intelligence reports.

It provides detailed insights into:

🧠 Tumor Type Distribution  
📍 Brain Region Analysis  
📈 Annual Diagnosis Trends  
👨‍⚕️ Age Group Analysis  
⚠️ Tumor Severity by Grade  
🩻 MRI Scan Visualization
